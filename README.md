# Business Performance & Risk Analytics Dashboard

## 📊 Overview
This project analyzes business performance by evaluating sales, profit, and risk across different regions and categories.  
The goal is to convert raw data into meaningful insights for better decision-making.

---

## 🛠 Tools & Technologies
- Excel (Data Cleaning)
- SQL (CTE, Window Functions)
- Power BI (Dashboard & Visualization)

---

## 🔄 Process
- Cleaned and prepared raw data in Excel  
- Performed data analysis using SQL  
- Used CTEs and Window Functions (LAG, RANK, SUM OVER)  
- Built interactive dashboard in Power BI  

---

## 📈 Key Insights
- South region has the most high-risk orders  
- West and North regions generate the highest sales  
- Furniture and Office Supplies bring in most of the revenue  
- Sales have gone down in recent months  
- High sales do not always mean high profit  

---

## 📁 Project Files
- Dataset (Raw & Cleaned)
- SQL Queries (`analysis.sql`)
- Power BI Dashboard (`.pbix`)

## 💡 Conclusion
This project demonstrates how data can be used to identify risks, track performance, and support business decisions.
